# 打印工作流设计规则

> 状态：当前设计参考。
> 当前用途：保留顶栏打印链路、文档规格、打印校准、打印检查和当次输出参数的产品规则。

## 背景

YiboLabel 的核心目标不是做一个通用排版工具，而是做一个本地优先、快速可靠的标签打印工作台。

理想工作流应明确区分两类操作：

- 半一次性设置：模板、词库、文档规格、打印机、打印校准。
- 当次打印操作：基于模板修改本次内容、用词库快速替换常用数据、手动修改不常用文本、调整打印份数、快速打印。

现有 `页面布局` 弹窗同时承载文档规格、打印校准、打印机选择、打印份数、风险提醒、保存和打印动作，导致一次性设置和当次打印参数混在一起。改造目标是拆清职责，让日常高频路径收敛为：

```text
选模板 -> 改本次内容 -> 改份数 -> 立即打印
```

## 总体结论

顶栏打印链路拆为以下形态：

```text
[文档规格]    [打印机 · 在线 HB-Q2] [打印校准 · 默认]    [份数 - 1 +] [立即打印][检查]
```

当当前状态不适合直接快速打印时，输出按钮组收敛为：

```text
[文档规格]    [打印机 · 在线 HB-Q2] [打印校准 · 未设置]    [份数 - 1 +] [检查并打印]
```

分组语义：

- `文档规格`：当前模板/草稿的画布基础。
- `打印机 + 打印校准`：当前设备及其输出适配。
- `份数 + 立即打印/检查`：当次输出。

原 `预览` 按钮删除。预览能力并入 `打印检查`。

## 设置分层

### 半一次性设置

这些设置通常只在建模板、换介质、换打印机、校准设备时调整，不应挡住日常快速打印。

- 模板。
- 词库。
- 文档规格。
- 打印机。
- 打印校准。

### 当次打印内容

这些是每次打印前可能发生的内容变化。

- 用词库替换常用文本、条码或二维码内容。
- 手动修改不常用文本。
- 针对当前模板做少量临时编辑。

### 当次输出参数

`打印份数` 属于当次输出参数，可能频繁改变，不应藏在检查页或设置页里。

推荐将份数常驻在顶栏打印组，靠近快速打印按钮：

```text
[份数 - 1 +]
```

数字区域可点击直接输入，后续可扩展常用份数快捷值，例如 `1 / 2 / 3 / 5 / 10`。

## 顶栏入口

### 文档规格

入口文案：

```text
[文档规格]
```

职责：

- 编辑当前模板/草稿的规格。
- 管理规格预设的选择、保存、复制、隐藏和归档。
- 只处理文档和模板的画布基础，不处理打印机、校准、份数和检查。

### 打印机

入口形态：

```text
[打印机 · 在线 HB-Q2]
```

职责：

- 显示当前目标打印机。
- 显示在线/离线状态。
- 支持切换当前文档目标打印机。
- 与 `打印校准` 紧邻，强调校准数据与当前打印机强关联。

### 打印校准

入口形态：

```text
[打印校准 · 默认]
[打印校准 · 已调整]
[打印校准 · 未设置]
```

职责：

- 编辑当前打印机的输出校准。
- 校准数据与当前打印机强关联。
- 切换打印机后，校准摘要应随之切换。

### 输出按钮组

正常可快速打印时：

```text
[立即打印][检查]
```

这是一个视觉上合并的分裂按钮：

- `立即打印` 是主按钮，占较大宽度。
- `检查` 是小按钮，占较小宽度。
- 两半紧挨，中间用细分隔线区分。

需要先检查或设置时：

```text
[检查并打印]
```

触发 `检查并打印` 的典型场景：

- 第一次打印当前文档。
- 当前打印机未选择。
- 当前打印机离线或不可用。
- 当前打印机没有校准记录。
- 当前文档规格发生变化后尚未检查。
- 打印参数发生变化后尚未检查。
- 存在条码或二维码重叠等高风险问题。
- 存在阻塞性内容问题。

## 文档规格

### 数据模型结论

文档规格直接作为模板的一部分保存。模板保存完整规格快照，而不是实时引用外部规格预设。

推荐模型：

```text
模板 = 文档规格 + 元素布局 + 词库绑定 + 默认输出参数
规格预设 = 可复用的尺寸方案，用于新建或套用到模板
```

规则：

- 每个模板必须始终保存完整文档规格。
- 文档规格可以来自规格预设。
- 模板保存规格快照，而不是实时绑定预设。
- 模板可记录 `sourceSpecId` 用于显示来源，但来源只是追踪，不是强绑定。
- 修改规格预设不自动影响已有模板。
- 用户可手动将某个规格预设应用到当前模板。
- 修改当前模板规格会让当前模板进入未保存状态。

### 规格预设生命周期

规格预设分为：

- 可选规格预设：新建模板或套用规格时默认可见。
- 历史规格快照：已有模板仍在使用，但不再推荐新建使用。

规则：

- 已被模板引用的规格预设不能物理删除。
- 已被引用的规格可隐藏或归档。
- 隐藏/归档后，新建和套用列表默认不显示，但旧模板仍能显示来源。
- 已被引用的规格修改尺寸时，默认走复制为新规格。
- 已被引用的规格允许安全编辑名称、备注、隐藏状态和排序。
- 没有任何模板引用的规格才允许真正删除。

当用户尝试修改已被引用的规格尺寸时，推荐提示：

```text
此规格已有 N 个模板使用。
建议复制为新规格后修改，避免影响历史来源记录。
```

### 文档规格面板内容

当前模板规格：

- 模板名称。
- 宽度。
- 高度。
- 间隙。
- 规格来源。
- 当前规格摘要，例如 `40 x 30 mm · 间隙 2 mm`。

规格预设：

- 选择规格预设。
- 保存当前规格为预设。
- 复制当前来源规格为新规格。
- 隐藏/归档规格。
- 查看隐藏/历史规格。

不放入文档规格：

- 打印机选择。
- 打印偏移。
- 打印浓度。
- 黑白反相。
- 打印份数。
- 重叠风险。
- 立即打印。

### 打开形式

文档规格复用主界面右侧框体：

```text
左侧元素栏 | 中间画布 | 右侧：文档规格
```

它不是弹窗，不打开新窗口。原因是规格会影响画布基础，用户需要一边看画布一边调整。

## 打印校准

### 定位

打印校准只管：

```text
这台打印机怎么把这张标签打准
```

它与当前打印机强关联。不同打印机的校准数据可能完全不同，切换打印机后必须显示对应打印机的校准状态。

### 面板内容

顶部摘要：

- 当前打印机名称。
- 在线/离线状态。
- 当前校准状态：`默认` / `已校准` / `未确认` / `未设置`。
- 当前文档规格摘要，例如 `40 x 30 mm · 间隙 2 mm`。

核心设置：

- 横向偏移 X。
- 纵向偏移 Y。
- 打印方向。
- 打印浓度。
- 黑白反相。

辅助动作：

- 测试打印。
- 重置为默认。
- 保存当前校准。
- 另存为当前打印机的校准方案。

说明和提醒：

- 当前校准仅适用于当前打印机。
- 切换打印机后使用对应打印机的校准。
- 如果当前打印机没有校准记录，显示正在使用默认校准。

不放入打印校准：

- 打印份数。
- 模板名称。
- 宽度、高度、间隙编辑。
- 重叠风险。
- 保存模板/另存模板。

### 打开形式

打印校准复用主界面右侧框体：

```text
左侧元素栏 | 中间画布 | 右侧：打印校准
```

它不是弹窗，不打开新窗口。这样测试打印后可以快速回到同一工作上下文继续微调。

## 打印检查

### 定位

打印检查合并原本的预览和检查能力：

```text
打印检查 = 打印预览 + 风险检查
```

它的目标是打印前最后确认：

- 会打成什么样。
- 能不能打。
- 哪里有风险。
- 需要去哪里修复。

### 打开形式

打印检查作为独立 `surface` 覆盖主工作区，类似模板库和词库管理页。

进入检查后，从编辑三栏：

```text
左元素栏 | 中画布 | 右属性栏
```

切换为检查两栏：

```text
左打印预览 | 右检查面板
```

不显示元素列表区。原因：

- 检查阶段不是管理元素顺序。
- 风险定位应直接显示在预览上。
- 保留元素列表会压缩预览空间。
- 右侧检查项已经承担问题导航职责。

### 左侧打印预览

显示：

- 当前标签最终输出预览。
- 按当前文档规格显示标签。
- 应用当前打印方向、偏移、反相等输出参数。
- 支持适合窗口和 100% 视图。

风险标注：

- 普通重叠：橙色区域框。
- 条码/二维码重叠：红色区域框。
- 空内容：虚线框。
- 超出画布：红色裁切线。
- 隐藏元素：不显示在预览上，只在右侧提示。

预览上只放标记，不放大段说明。具体说明放右侧检查面板。

### 右侧检查面板

输出摘要：

- 当前文档名称。
- 当前规格。
- 当前打印机。
- 当前校准。
- 当前份数。
- 保存状态。

检查项：

- 是否已选择打印机。
- 打印机是否可用。
- 是否有当前打印机的校准。
- 文档规格是否有效。
- 元素是否超出画布。
- 是否存在元素重叠。
- 是否存在条码/二维码重叠。
- 是否有空文本、空条码、空二维码。
- 是否有隐藏元素。
- 当前文档是否有未保存修改。

检查项状态：

- `通过`。
- `提醒`。
- `阻塞`。

阻塞项示例：

- 没有可用打印机。
- 未选择打印机。
- 文档规格无效。
- 条码或二维码内容为空。
- 打印接口不可用。

提醒项示例：

- 当前打印机未校准，正在使用默认校准。
- 普通元素重叠。
- 条码或二维码重叠。
- 存在隐藏元素。
- 当前文档有未保存修改。

底部动作：

- 返回编辑。
- 打开文档规格。
- 打开打印校准。
- 保存。
- 立即打印。

点击 `打开文档规格` 或 `打开打印校准` 时，退出检查 surface，回到编辑 surface，并打开对应右侧面板。

## 暂不处理

- 多打印机复杂校准继承。
- 自动校准向导。
- 条码/二维码可扫性自动检测。
- 位图一致性打印模式。
- 规格预设批量迁移模板。
- 多窗口打印检查。
